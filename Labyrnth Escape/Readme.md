# Labyrinth Escape

*Ein selbst entwickeltes Spielprojekt mit Python und Arcade*

## Worum geht es in diesem Projekt?

In diesem Projekt habe ich ein eigenes **Labyrinth-Spiel** programmiert.
Der Spieler steuert eine Figur durch ein Spielfeld, sammelt Münzen ein und versucht, das Ziel zu erreichen, ohne dabei gefährliche Blöcke zu berühren.

Das Spiel verbindet **logisches Denken**, **Reaktionsfähigkeit** und **Orientierung**, da falsche Bewegungen direkt zum Game Over führen können.



## Warum ich dieses Projekt gemacht habe

Ich wollte ein Projekt umsetzen, bei dem ich:

* ein richtiges Spiel von Anfang bis Ende entwickle
* lerne, wie Spielfluss, Steuerung und Zustände zusammenhängen
* meine Python-Kenntnisse praktisch anwende

Besonders wichtig war mir, dass das Spiel **nicht nur technisch funktioniert**, sondern sich auch wie ein echtes Spiel anfühlt.



## Spielprinzip

Der Spieler startet an einem festen Punkt im Labyrinth und bewegt sich mit den Pfeiltasten.

Im Spiel gibt es:

* **Wände**, die das Labyrinth bilden
* **gefährliche Blöcke**, die sofort zum Game Over führen
* **Zielblöcke**, die das Spiel erfolgreich beenden
* **Münzen**, die eingesammelt werden können

Zusätzlich läuft ein **Timer**, sodass man sehen kann, wie lange man für das Labyrinth gebraucht hat.



## Steuerung

* Pfeiltasten: Bewegung
* SPACE: Spiel starten
* ENTER: Neustart nach Gewinn oder Game Over

Die Steuerung ist bewusst einfach gehalten, damit der Fokus auf dem Spiel selbst liegt.



## Spielablauf

* Beim Start erscheint ein Startbildschirm
* Danach beginnt das eigentliche Spiel
* Berührt man einen gefährlichen Block, ist das Spiel sofort vorbei
* Erreicht man das Ziel, gewinnt man das Spiel
* Am Ende werden **Zeit** und **gesammelte Münzen** angezeigt

So bekommt man direkt Feedback über seine Leistung.
